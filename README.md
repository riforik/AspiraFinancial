## 📄 **Getting Started**

### **Prerequisites**  
1. Node.js >= 14.x  
2. NPM or Yarn  

### **Installation**  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/riforik/aspira.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd aspira  
   ```  
3. Install dependencies:  
   ```bash  
   npm install  
   ```  
4. Start the development server:  
   ```bash  
   npm run dev  
   ```
